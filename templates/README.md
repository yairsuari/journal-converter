# Templates

Journal-specific Word reference docs (`.docx`) and LaTeX class files (`.cls`) go here.

**Licensing policy:** only bundle a file if its license is permissive (LPPL, MIT, or
equivalent). Check the license header in `.cls` files before committing. If a journal's
template has no stated license or a restrictive one, leave it out — the converter will
warn the user and fall back to default styling, and the journal's config should note
where to download it from.
